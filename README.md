# Library Database Management System

A simple Python + MySQL based Library Database Management System for handling books, members, and book issue/return records.

## Features
- Add new books
- Register members
- Issue books to members
- Return issued books
- View all books and members

## Tech Stack
- Python
- MySQL

## How to Run

1. Install MySQL and create database using `database.sql`
2. Install requirements:
   pip install -r requirements.txt
3. Run the program:
   python app.py

## Project Info
This project demonstrates database handling, SQL queries, and basic CRUD operations using Python.
